# Quiet Space - 深夜陪伴式AI日记空间

一个让Z世代学生在安静的夜晚与AI朋友聊天，AI帮你把对话变成日记的Web应用。

## 🎯 项目简介

**一句话描述**：深夜陪伴式AI日记空间，让Z世代学生在安静的夜晚与AI朋友聊天，AI帮你把对话变成日记，并长期记住你的故事。

**核心理念**：打造一个不喧闹、不功利、有温度的数字陪伴空间。用户打开它不是为了"完成任务"，而是为了"被理解、被记住"。

## 🚀 快速开始

### 启动服务器

```bash
cd quiet-space
node server.js
```

服务器将在 `http://localhost:8080` 启动。

### 访问应用

在浏览器中打开 `http://localhost:8080`

## ✨ 核心功能

### 1. 倾诉（Chat-to-Diary）
- 与AI对话，AI自动生成日记草稿
- 3D粒子时间线展廊浏览历史对话
- 支持上传图片作为对话背景

### 2. 回忆（Diary Browser）
- 浏览和编辑AI生成的日记
- 按情绪、时间筛选日记
- 双向关联原始对话

### 3. 碎碎念（Fragment）
- 记录零碎想法
- AI帮你组织成完整日记
- 快速记录，不打断思路

## 🛠️ 技术栈

- **前端**：HTML5 + CSS3 + JavaScript
- **后端**：Node.js
- **3D效果**：Three.js 粒子系统
- **AI接口**：OpenAI兼容API

## 📝 设计原则

1. **粒子系统是核心视觉层** - 整个产品基于Three.js粒子系统构建视觉语言
2. **页面过渡必须有视觉过渡** - 所有页面切换都需要视觉过渡动画
3. **氛围优先于功能** - 功能性UI不能破坏页面的沉浸感
4. **深色、安静、有温度** - 所有视觉设计的底线

## 🎨 页面入口

首页三个主入口，用户选择"此刻想做什么"：

| 入口 | 功能 | 映射页面 |
|------|------|---------|
| 📖 回忆 | 看历史日记 | diary.html |
| 💬 倾诉 | 跟AI对话 | chat.html |
| 🧩 碎碎念 | 记录零碎想法 | fragment.html |
